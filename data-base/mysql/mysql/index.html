      MySQL

      mysql 应用广泛，并不是每个人都有条件用上 navicat 的。你需要了解 mysql 的连接方式和基本的操作，在异常情况下才能游刃有余。

      mysql -u root -p -h 192.168.1.2
      # 查看操作系统版本
      sudo lsb_release -a
      # 查看本机 IP
      ipconfig
      # 查看本机端口
      netstat -ano
