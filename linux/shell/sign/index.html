    管道和重定向

    管道与管道符

    • 管道和信号一样，也是进程通信的方式之一
    • 匿名管道（管道符）是 Shell 编程经常用到的通信工具
    • 管道符是 |，将前一个命令执行的结果传递给后面的命令
    ps | cat
    echo 123 | ps

    重定向符号

    一个进程默认会打开标准输入、标准输出、错误输出三个文件描述符。

    输入重定向符号：<

    # 先编辑文本文件，并随便输入字符后保存
    vim a.txt
    # 将 a.txt 读入到 var2 变量中，代替手动输入
    read var2 < a.txt
    # 输出 var2 变量，打印出 a.txt 内容
    echo $var2

    输出重定向符号：>、>>、2>、&>

    • >：相当于 1>，代表从左到右的重定向方向
    • >>：代表追加，也就是不改变重定向目标文件原有的内容，追加在后面
    • 2>：错误重定向

    接上例子

    # 将变量 var2 值输出到 b.txt
    echo $var2 > b.txt
    # 查看 b.txt 验证
    cat b.txt
